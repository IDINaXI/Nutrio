АННОТАЦИЯ

Дипломный проект на тему «Web-приложение для персонального контроля здоровья Nutrio» включает в себя программный продукт, пояснительную записку и графическую часть.

Разработанное программное средство предназначено для комплексного ведения персональных данных о здоровье пользователя: отслеживания веса, объёмов тела, приёма лекарств, а также формирования отчётов для врача. Приложение обеспечивает удобный и интуитивно понятный интерфейс для пользователей, а также автоматизацию рутинных задач, связанных с мониторингом состояния здоровья.

Программная часть реализована с использованием современных технологий: серверная часть построена на платформе Java Spring Boot, клиентская часть — на фреймворке Flutter. Для хранения и обработки данных используется реляционная база данных, интегрированная с сервером. В приложении реализованы функции генерации PDF-отчётов, визуализации данных в виде графиков, а также система напоминаний о приёме лекарств.

Пояснительная записка содержит подробное описание архитектуры приложения, обоснование выбора технологий, описание пользовательских сценариев, а также результаты тестирования. Общий объём пояснительной записки составляет 146 страниц, она включает четыре основных раздела и два приложения.

Графическая часть проекта представлена на двух листах и содержит диаграмму последовательности и диаграмму вариантов использования, иллюстрирующие основные процессы взаимодействия пользователя с системой.
